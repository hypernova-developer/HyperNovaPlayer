# 🌌 HyperNova Player

**HyperNova Player** is a sleek, high-performance web-based music player designed with a minimalist cyberpunk aesthetic. It provides a seamless local audio playback experience with a focus on speed, precision, and visual immersion.

## ⚡ Features

* **Sonic Arsenal:** Batch upload local audio files directly into your browser.
* **Precision Seeking:** Interactive progress bar allows you to jump to any second of the track instantly.
* **Stepped Volume Control:** A 5-stage vertical volume slider for tactical audio adjustments.
* **Cyberpunk UI:** Dark-themed interface featuring a purple grid background and neon accents.
* **Interactive Cursor:** Custom-built reactive cursor dot for an immersive navigation experience.
* **Smart Playlist:** Automatic track switching and high-visibility active track indicators.
* **Loop Mode:** Continuous playback support for single tracks.

## 🛠️ Built With

* **HTML5 & CSS3:** Utilizing advanced gradients, backdrop filters, and grid layouts.
* **Vanilla JavaScript:** Lightweight and fast, managing the Web Audio API without external dependencies.
* **Edge Tools Optimized:** Fully compliant with accessibility (Axe API) and cross-browser standards.

## 🚀 Quick Start

1.  Clone the repository:
    ```bash
    git clone [https://github.com/eymen/HyperNovaPlayer.git](https://github.com/hypernova-developer/HyperNovaPlayer.git)
    ```
2.  Ensure you have your `favicon.png` in the root directory.
3.  Open `index.html` in any modern browser.
4.  Upload your music and ignite the engine.

HyperNova Player is part of the **SyntaX** project library under the `bettercpp` ecosystem. It follows the philosophy of "Pure Power" (CompileX version), focusing on performance and core functionality over unnecessary "cute" decorative elements.
